KATA PENGANTAR

Puji syukur saya sampaikan kepada Tuhan Yang Maha Esa, atas berkah, rahmat, dan karunia-Nya sehingga saya dapat menyelesaikan penyusunan web ini tepat pada waktunya. Web ini memuat tiga pokok materi geometri sebagai berikut :

  1. Garis Transversal, Segitiga, dan Segiempat
  2. Teorema Phytagoras
  3. Transformasi Geometri Tunggal

Penyusunan web ini diperuntukkan bagi pembaca yang mengikuti pembelajaran matematika pada tingkat SMP sederajat pada materi Geometri. Tujuan penyusunan web ini adalah untuk memenuhi tugas mata kuliah TIK untuk Pembelajaran serta untuk menambah pengetahuan pembaca dalam bidang matematika terkait materi Geometri. Ucapan terima kasih kami peruntukkan kepada Bapak Dr. Aan Hendrayana, S.Si., M.Pd. selaku dosen pengampu mata kuliah TIK untuk Pembelajaran yang telah membimbing saya dalam proses pembuatan web ini, serta kepada diri saya sendiri yang telah membuat web ini. Semoga web ini dapat bermanfaat untuk creator dan bagi para penggunanya.
